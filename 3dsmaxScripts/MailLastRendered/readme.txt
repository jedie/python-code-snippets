 Thunderbird Mailer

 Save the last rendered picture as jpeg and compose a mail with thunderbird

 Tested with MAX 2009
______________________________________________________________________________

 To install this Script:
    1. Drag&Drop the MAXScript ZIP Package (*.mzp File) to a runing 3dsmax
    2. goto 'Customize/Cusomize User Interface/Toolbars'
    3. goto Category: 'htFX Scripts'
    4. Drag and Drop the Action to you Toolbar

 more Info: "MaxScript Reference/MacroScripts for New Users"
